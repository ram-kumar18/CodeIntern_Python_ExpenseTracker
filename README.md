# Personal Expense Tracker

A Python console application to track your daily expenses, view summaries, and visualize spending using charts.

## Features

- Add expenses (date, category, description, amount)
- View daily, weekly, and monthly expense summaries
- See top spending categories
- Visualize spending using bar and pie charts (Matplotlib)
- Store data locally in `expenses.csv`

## Technologies Used

- Python 3
- pandas
- matplotlib
- CSV for data storage
